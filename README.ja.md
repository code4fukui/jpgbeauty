# jpgbeauty

大きな JPEG ファイルをウェブ公開用に圧縮・リサイズできる便利なツールです。ファイルをドラッグ&ドロップするだけで、最大サイズや画質を調整して ZIP ファイルでダウンロードできます。サーバーにデータをアップロードする必要がありません。

## 機能
- JPEG や PNG ファイルのリサイズと圧縮
- 複数ファイルの一括変換と ZIP ダウンロード  
- 最大サイズ、画質、sRGB色空間の設定

## 使い方
1. JPEG または PNG ファイルをブラウザ上にドロップします。
2. 最大幅/高さ、画質、sRGB 強制の設定を調整します。
3. 変換されたファイルが ZIP アーカイブでダウンロードされます。
